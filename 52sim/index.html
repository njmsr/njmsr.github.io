Reading the variables

Variable 1 decides the shape of each parent’s 5-run: the Wed/Thu holder gets Wed→Sun (the weekend lands inside their run); the Mon/Tue holder gets Fri→Tue (their run wraps the weekend into the next week).

Variable 2 decides the phase — which weeks are whose. It changes nothing structural; it only slides the alternation by one week.

In every configuration: Saturdays split evenly, handoff seams stay fixed (Sun→Mon, Tue→Wed, Thu→Fri on swap weeks), and nobody negotiates anything after the two switches are set.
