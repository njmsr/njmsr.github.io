How it works

Weekday roles are fixed every week: Stefani takes Mon, Tue, Thu; Nathan takes Wed, Fri, Sat. The single variable sets which parent opens the Sunday rotation; Sundays alternate from there.

The longest stretch either parent holds is three days — the run the Sunday attaches to. Pickup and drop-off times are set separately.
